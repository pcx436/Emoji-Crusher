# Emoji Crusher
Candy crush clone but with emoji.

## Instructions to Run:
```
cd src/com/emojiCrusher
javac -cp ".:../../../lib/*" *.java
cd ../../
java -classpath "../lib/*:." com.emojiCrusher.Controller <PATH_TO_EMOJI_PNG_DIRECTORY>
```
